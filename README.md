# 🎓 Student Enrollment System

A simple React-based web application to manage student records with full CRUD operations using a mock backend.

---

## 🚀 Features

- ➕ Add new students  
- 📋 View student list  
- 🔍 Search students by name  
- 🗑️ Delete student records  
- 🔔 Toast notifications for actions  
- 🌐 API integration using json-server  

---

## 🛠️ Tech Stack

- Frontend: React (Vite)
- Routing: React Router
- Backend: json-server
- Notifications: React Toastify

## ⚙️ Installation & Setup

### 1. Clone the repository
git clone https://github.com/YOUR-USERNAME/student-enrollment-system.git

### 2. Navigate to project
cd student-enrollment-system

### 3. Install dependencies
npm install

### 4. Start backend (json-server)
json-server --watch db.json --port 3000

### 5. Run frontend
npm run dev
